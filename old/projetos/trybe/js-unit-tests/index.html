Sobre o Projeto

O Projeto JavaScript Unit Tests consistiu na aplicação dos aprendizados durante o bloco sobre Testes Unitários por meio do Jest, um framework utilizado para desenvolver testes unitários para funções JavaScript.

Experiência Pessoal

O projeto JavaScript Unit Tests foi um desafio para mim. Todos os projetos anteriores não necessitavam que nós, pessoas estudantes, implementássemos os testes unitários dos requisitos. Neste projeto em particular fomos colocados em três situações para que pudéssemos compreender a importância dos testes no dia a dia de uma pessoa desenvolvedora.

  • Implementação de testes unitários para funções pré-desenvolvidas pela equipe da Trybe
  • Implementação de funções para testes unitários pré-desenvolvidos pela equipe da Trybe
  • Implementação de funções e testes unitários para um sistema de comanda de restaurante

Habilidades Desenvolvidas

Dentre as habilidades desenvolvidas durante a execução do projeto as seguintes foram as que eu mais desenvolvi:

  • Conceitos de Desenvolvimento Orientado a Testes (TDD)
  • Curva de aprendizagem do Framework Jest
  • Implementação de Testes Unitários em JavaScript
  • Implementação de Funções em JavaScript
  • Estudo e Implementação de um sistema de comandas de restaurante
