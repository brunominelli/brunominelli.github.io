Sobre o Projeto

O Projeto Playground Functions consistiu no desenvolvimento de uma série de requisitos que deveriam ser reproduzidos em código de funções por meio do uso da linguagem de programação JavaScript.

O objetivo do projeto era consolidar os conhecimentos adquiridos até no Curso de Desenvolvimento Web Fullstack da Trybe a fim de garantir que as pessoas estudantes estavam aptas à seguir para o próximo bloco do curso.

Experiência Pessoal

A minha experiência com o projeto Playground Functions foi tranquila apesar de eu ter tido algumas dificuldades para implementar alguns dos requisitos obrigatórios e bõnus. Todavia, consegui realizar a entrega do que havia sido requerido dentro do prazo estipulado com 100% dos requisitos totais do projeto.

Ainda sobre os requisitos do projeto, muitas das funções implementadas durante sua execução foram baseadas em conceitos aprensentados anteriormente nas aulas ministradas e no conteúdo disponibilizado às pessoas estudantes para estudo, pesquisa e referências.

Habilidades Desenvolvidas

Dentre as habilidades desenvolvidas durante a execução do projeto as seguintes foram as que eu mais desenvolvi:

  • Lógica de programação
  • Baby Steps: Implementação passo a passo
  • Sintaxe JavaScript
  • Identação de código fonte
  • Nível de Complexidade de Funções
