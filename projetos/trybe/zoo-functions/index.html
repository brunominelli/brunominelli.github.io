Sobre o Projeto

O Projeto Zoo Functions consistiu na aplicação dos aprendizados durante o bloco sobre Higher Order Functions do JavaScript ES6+ desenvolvendo um sistem de relatórios para um zoológico.

Experiência Pessoal

O projeto Zoo Functions foi um desafio para mim porque os requisitos não estavam muito bem escritos e aplicar exclusivamente as HOF's em cada um dos requisitos foi algo extremamente trabalhoso. Creio que este foi o projeto mais complexo a nível de compreensão das regras de negócios.

Habilidades Desenvolvidas

Dentre as habilidades desenvolvidas durante a execução do projeto as seguintes foram as que eu mais desenvolvi:

  • Conceitos de Desenvolvimento Orientado a Testes (TDD)
  • Implementação de Funções em JavaScript
  • Estudo e Implementação de um sistema de comandas de restaurante
